# topclass-homepage

이 저장소는 로컬에서 실행 중인 `http://127.0.0.1:3000` 사이트를 Vercel로 배포하기 위한 최소 정적 사이트 템플릿입니다.

배포 방법 요약:

- Vercel 웹사이트에서 `Import Project` → GitHub 연동 후 이 저장소 선택 → 기본 설정으로 배포하세요.

파일 구성:

- `index.html`: 메인 페이지
- `vercel.json`: Vercel 빌드/라우팅 설정
- `.gitignore`: 일반 항목

환경 변수 없이 기본 설정으로 바로 Deploy 가능합니다.
